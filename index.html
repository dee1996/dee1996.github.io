关于链表

发表于 2018-03-21

摘要:

- 经常在leetcode上看到链表操作，之前也研究过，但是模模糊糊的没有什么深刻的印象，今天记录一下，方便以后查找

正文:

- 首先了解链表的结构：
-----     ------------        ------------

|head|—>|data|index|–…–>|data|null|
—– ———— ————
有一个head，里面不存储数据，只有指向下一个数据的索引，下一个索引中存储有数据以及再下一个数据的索引，一直到最后一个数据的索引指向null。

首先需要写一个链表的实体类，代码如下：
public class Node {
public Node next; // 代表下一个节点
public int data; //储存数据
public Node(int data) {
this.data = data;
}
}

对链表的一些操作：
public class LeetCode2 {
private Node head;
public static void main(String[] args) {
LeetCode2 code = new LeetCode2();
code.addNode(2);
code.addNode(3);
code.addNode(4);
code.addNode(5);
code.printNode();
}
/**

 * 添加节点
 * @param data
 */
public void addNode(int data) {
    Node newNode = new Node(data);
    if(head == null) {
        head = newNode;
        return;
    }
    Node temp = head;
    while(temp.next != null) {
        temp = temp.next;
    }
    // 将数据添加到最后一个节点
    temp.next = newNode;
}
/**
 * 打印链表最后输出2345
 */
public void printNode() {
    while(head != null) {
        System.out.print(head.data);
        head = head.next;
    }
}

}
这里我有一个疑问，按理head节点应该是不存放数据的，但是我目前的这种写法，head节点是第一个节点，也就是说没有head节点，
直接从第一个有数据的节点开始了，这里先留着，日后了解其中原由再修改。

LeetCode:Two Sum

发表于 2018-03-12

摘要:
LeetCode第一题:给定一个数组及目标值，查找出这个数组中两个数相加之和等于目标值的下标
正文:
Example:
Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

分析:
遍历数组，找到两个值，如果找到这两个值就返回，找不到则返回null
代码:
public static int[] twoSum(int[] arr,int num) {
for(int i = 0;i < arr.length - 1;i++) {
for(int j = i + 1;j < arr.length;j++) {
if(arr[i] + arr[j] == num) {
return new int[] {i,j};
}
}
}
return null;
}
