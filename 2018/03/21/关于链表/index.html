关于链表

发表于 2018-03-21

摘要:
经常在leetcode上看到链表操作，之前也研究过，但是模模糊糊的没有什么深刻的印象，今天记录一下，方便以后查找
正文:
首先了解链表的结构:

|head|—>|data|index|–…–>|data|null|

有一个head，里面不存储数据，只有指向下一个数据的索引，下一个索引中存储有数据以及再下一个数据的索引，一直到最后一个数据的索引指向null。

  • 首先需要写一个链表的实体类，代码如下:

    public class Node {
public Node next; // 代表下一个节点
public Object data; //储存数据
public Node() {
}
public Node(Object data) {
    this.data = data;
}
}

  • 对链表的一些操作：

    public class LeetCode2 {
private Node head = new Node(); // 实例化一个Node为head节点
public static void main(String[] args) {
    LeetCode2 code = new LeetCode2();
    code.addNode(2);
    code.addNode(3);
    code.addNode(4);
    code.addNode(5);
    code.printNode();
}
/**
* 添加节点
* @param data
*/
public void addNode(int data) {
    Node newNode = new Node(data);
    Node temp = head;
    while(temp.next != null) {
        temp = temp.next;
    }
// 将数据添加到最后一个节点
    temp.next = newNode;
}
/**
* 打印链表最后输出2345
*/
public void printNode() {
    while(head.next != null) {
        System.out.print(head.next.data);
        head = head.next;
        }
    }
}
